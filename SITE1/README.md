# Rushil Kumar — Professional Portfolio

A stunning enterprise-meets-edgy portfolio site built with **Next.js 15**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Production Build

```bash
npm run build
npm start
```

## Sections

- **Hero** — Introduction with resume download
- **About** — Professional summary and key stats
- **Career Journey** — Interactive timeline from Deeplogictech to DuQuode
- **Skills & Education** — Technical expertise and academic background
- **Portfolio** — Placeholder cards for upcoming case studies and projects
- **Contact** — Email, WhatsApp, location, and resume links
